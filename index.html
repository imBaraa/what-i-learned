HTML, Tags:

Tag What it does Does it have a closing? Attributes
a linking text with a URL Yes href="URL HERE" and Global Attributes
b bold text, with no meaning of importance Yes Global Attributes
br line breaker, line down No Global Attributes
center centers anything in the middle of the page Yes None
div making a division in the page Yes CSS and Global Attributes
em sloped text, with meaning of importance Yes Global Attributes
form creates a form that does an action Yes action="mailto:myemail@gmail.com" or action="index.html" enctype="text/plain" and Global Attributes
h1 Heading1, largest size Yes Global Attributes
h2 Heading2, less than Heading1 Yes Global Attributes
h3 Heading3, less than Heading2 Yes Global Attributes
h4 Heading4, less than Heading3 Yes Global Attributes
h5 Heading5, less than Heading4 Yes Global Attributes
h6 Heading6, less than Heading5 Yes Global Attributes
hr horizontal line No size="5", noshade, color, width, CSS styles, Global Attributes
i sloped text, with no meaning of importance Yes Global Attributes
img display image No alt="write something here if photo doesn't appear", src="URL or Location here", width height, Global Attributes
input input that can be text, email, button, submit No name="write here something that you want to identify it later in submission", type="text", type="submit", type="button", Global Attributes
li listing items in ordered or unordered lists Yes type="a", type="A", type="i", type="I", type="1", Global Attributes
link linking an external stylesheet (CSS), or website icon Yes rel="stylesheet", rel="favicon", href="URL here, or location", Global Attributes
label label some text Yes Global Attributes
ol creates an ordered list {1,2,3} Yes start="from which number you want to start", Global Attributes
p paragraph Yes Global Attributes
strong bold text, with meaning of importance Yes Global Attributes
style making an internal (CSS) in head Yes Global Attributes
textarea creates a text box that can hold a lot of text and has adjustable size Yes rows="number of starting rows", cols="number of starting cols", Global Attributes
table creates a table Yes bgcolor="red", border="2", cellspacing="10", cellpadding="5", Global Attributes
tr table row Yes bgcolor="red", Global Attributes
td table cell Yes bgcolor="red", Global Attributes
th makes a head cell, specifying whats underneath Yes bgcolor="red", Global Attributes
ul creates an unordered list {⚫,⚫,⚫} Yes type="circle", type="disc", type="square", Global Attributes

This is a very brief information, for the full information visit MDN and search by a certain tag, or visit devdocs and search by a certain tag.

CSS

CSS is the shortcut for: Cascading Style Sheet

There are three types of CSS:

  1. inline CSS
  2. internal CSS
  3. External CSS

1-inline CSS:

is the most specific among the three types along with the tag Attributes, and it is written inside the tag brackets like this:

<p style="font-size: 100px;">here is your text</p>
